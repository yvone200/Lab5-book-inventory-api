# Book Inventory API - Lab 5

## Overview
This project is a *Book Inventory API* developed using *FastAPI* and *SQLModel*. The API provides functionalities for managing book records, including creating, viewing, searching, updating, and deleting books.

## Technologies Used
- Python
- FastAPI
- SQLModel
- SQLite Database
- Uvicorn

## Features
The API supports the following operations:

- Add new books
- View all books
- Search books
- Retrieve a book by ID
- Update book details
- Partially update book information
- Delete books
